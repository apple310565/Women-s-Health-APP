# 大學畢業專題: 中醫婦科的辨證與應用
以Android Studio 作為開發環境,使用 Java 語言撰寫的女性養生APP，包含使用模糊集合
去計算症候歸屬度的婦科辨證功能，與紀錄使用者健康狀況，提供飲食習慣及運動穴道按摩等相關
建議的健康促進功能。
